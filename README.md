# Deltarune-save-mod
ever wanted more than 3 save slots for deltarune? Now you can have up to 6,442,450,941 slots! (6.4 billion)

The windows batch script is only to be used for setting up your installation for the patches. If you do not understand what the code does, just do it yourself!

This mod will not affect your default instalation's save system. The formaat is "filech*chapter*_*slot*". If it is on page 2 or higher, "filech*chapter*_*slot*_*page number-1*"

this mod is compatible with https://github.com/Kitty-999ooo/Kris-Completion-Reverser
